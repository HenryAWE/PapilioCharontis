# 自定义构建
## CMake 选项
|           选项            |  类型  |                   解释                   |
| :-----------------------: | :----: | :--------------------------------------: |
|  `papilio_build_example`  | `BOOL` |                 构建示例                 |
| `papilio_build_unit_test` | `BOOL` |               构建单元测试               |
|  `papilio_build_module`   | `BOOL` | 构建 C++ 20 模块（module）支持（实验性） |
|   `papilio_c_interface`   | `BOOL` |               构建 C 接口                |
